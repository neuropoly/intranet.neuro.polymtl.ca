Manual method ¬∂

https://www.virtualbox.org/manual/ch04.html#idp91444064 If you want to transfer files between your computer (host) and your Virtual Machine (guest) you‚Äôll need a shared folder.

  • Create a folder on your computer that will be your shared folder e.g. /home/user/VM_shared

  • Go to VirtualBox and open the Settings of your VM

  • Enter the path to your folder AND give it a different name e.g VBShared.

  • Now, start your VM and create a folder e.g /home/user/Shared_Folder

  • Now you need to edit the file /etc/rc.local and add this above the ‚Äúexit 0 ‚Äù line: mount -t vboxsf -o username=uid, groupname=gid VBShared /home/user/Shared_Folder

      • gid: getent group groupname (e.g. groupename = NeuroPoly, the command will give NeuroPoly:x:1000 where the 3rd argument is the gid)

      • uid: id -u username

    • For admin rights use sudo vi /etc/rc.local. Or su root. Then vi /etc/rc.local You need the admin password.

  • Now reboot your VM.

  • Make the change permanent: Go to VirtualBox‚Äì>settings‚Äì>foldershared‚Äì>double click‚Äì>‚ÄôMake Permanent‚Äô option

If that didn‚Äôt work it could be because the VBoxGuestAdditions are not installed on your VM. If so, follow those next step :

With a Windows Guest, download SDelete and run this:

sdelete ‚Äìz

Google Drive

Google Drive#

If you are a student, please create a Google Drive folder and share it with your supervisor. Here is an template you can copy. The folder should include the following:

Folder

Description

Name - Progress Report

Google doc listing progress report.

Admin/

administrative stuff such as Polytechnique documents, grades, CV

Biblio/

not ALL your biblio, but some interesting stuff that we exchange temporarily

Conferences/

subfolder prefix is the date of the event, e.g. 20150530_ISMRM/

Fellowships/

subfolder prefix is the date of the deadline, e.g. 20150314_FRQS/

Manuscripts/

subfolder name is year of first submission + journal, e.g. 2015_neuroimage. Inside the folder, please follow this organization.

Reports/

Include progress reports (format: Google Slides, created directly from this folder). Prefix name should start with YYYYMMDD_. Example: 20170515_ArtefactDetection.pptx

Results/

Snapshots/results of your experiments. Prefix should start with YYYYMMDD. Example: 20150120_dmri_nogse/

Note

For students doing an internship, you will only need to create the admin, biblio, and reports folders.

Date format should always be YYYYMMDD (or YYYY-MM-DD). See this: https://xkcd.com/1179/
